# NayePankh Foundation Website Prototype
A responsive website prototype designed and developed by me for NayePankh Foundation as part of a Web Development Internship assignment.
# Overview
This project was created to showcase how a modern and engaging website can help NayePankh Foundation increase volunteer participation and effectively communicate its mission.

The website focuses on simplicity, accessibility, and responsive design while providing visitors with information about the organization and opportunities to get involved.
# Project Objectives
Improve the online presence of NayePankh Foundation
Promote volunteer participation
Showcase the foundation's impact
Create a clean and user-friendly experience
Develop a responsive website accessible across devices

# Features
1. Hero Section

A visually appealing landing section highlighting the mission and vision of the foundation.

📚 About Us Section

Provides an overview of the organization and its purpose.

📊 Impact Statistics

Displays key metrics to demonstrate the foundation's contribution and reach.

🤝 Volunteer Registration Form

Allows visitors to express their interest in becoming volunteers.

📞 Contact Footer

Contains essential contact information and website branding.

📱 Responsive Design

Optimized for desktops, tablets, and mobile devices.
